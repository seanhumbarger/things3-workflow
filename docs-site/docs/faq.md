# Frequently Asked Questions (FAQ)

## How do I view plugin logs and errors?
To see detailed log messages or errors:
- Open Obsidian.
- Press `Cmd+Option+I` (macOS) or `Ctrl+Shift+I` (Windows/Linux) to open the Developer Console.
- Go to the **Console** tab to view plugin logs and errors.

## Can I use this on mobile?
No, desktop only.

## Does it support recurring tasks?
Yes, as long as they match your filters.

## Where is the cache stored?
In the plugin folder as `imported.json`.

## How do I clear the cache?
Use the command palette command.

## How do I update the plugin?
Update via Obsidian Community Plugins or manually replace the plugin files in your vault.

## Can I customize the note format?
Yes, to a degree. You can change note headers and structure in the plugin settings. You can also add custom tags and choose which metadata to include.

## Does this plugin write back to Things3?
No, this plugin does not write back to Things3. It is not bidirectional. This is a deliberate design decision to avoid modifying your source data in Things3.
