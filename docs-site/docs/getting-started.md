# Getting Started

## Prerequisites
- Obsidian v0.15.0 or higher
- Things 3 (macOS)

## Installation

### Community Plugins (Recommended)
1. Open Obsidian.
2. Go to **Settings > Community plugins**.
3. Search for "Things3 Workflow".
4. Install and enable the plugin.

### Manual Installation
1. Download the latest release from GitHub.
2. Extract files to `<vault>/.obsidian/plugins/things3-workflow/`.
3. Reload Obsidian and enable the plugin.

## Quickstart
1. Configure the plugin settings (see [Configuration](configuration.md)).
2. Use the command palette to trigger an import.
3. Imported tasks will appear as notes in your vault.
